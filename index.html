Qué hacemos

Una solución clara para ahorrar energía en casa

Nuestra empresa desarrolla un sistema inteligente de planificación energética doméstica capaz de transformar información compleja del sistema eléctrico en recomendaciones prácticas para cualquier hogar.

⚡

Analizamos precios horarios

Utilizamos datos energéticos oficiales para identificar las franjas más convenientes desde el punto de vista económico.

🏠

Modelamos tareas domésticas

Estudiamos el uso de electrodomésticos comunes como lavadora, lavavajillas, horno o termo eléctrico.

🕒

Generamos recomendaciones

Proponemos planes diarios de uso energético para ayudar al usuario a tomar decisiones más eficientes.

Cómo funciona

Del dato bruto a una recomendación útil

El sistema sigue un proceso estructurado que combina análisis de datos, modelización y optimización.

01

Recopilación de datos

Se obtienen precios horarios oficiales y previsiones meteorológicas desde fuentes públicas y verificables.

02

Tratamiento y almacenamiento

Los datos se normalizan y se organizan para su consulta, comparación y uso posterior dentro del sistema.

03

Optimización del consumo

Un algoritmo evalúa las ventanas horarias disponibles y selecciona las opciones más eficientes según cada tarea.

04

Plan diario y notificación

El usuario recibe recomendaciones comprensibles, estimaciones de ahorro y una planificación energética lista para aplicar.

Equipo del proyecto

Este proyecto busca integrar de forma práctica conocimientos adquiridos durante la formación universitaria, especialmente en análisis de datos, optimización y desarrollo de sistemas informáticos. A través de su planteamiento se pretende mostrar cómo las herramientas matemáticas y tecnológicas pueden transformar información compleja en recomendaciones útiles para el usuario. De este modo, el trabajo refleja el potencial de la ingeniería para abordar retos actuales relacionados con la eficiencia energética y la gestión responsable de los recursos.

 

A continuación se presenta el equipo responsable del desarrollo del proyecto. Cada integrante aporta una perspectiva complementaria en el ámbito técnico, analítico y aplicado.
